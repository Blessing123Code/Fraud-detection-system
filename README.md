# Fraud-detection-system
Machine learning model for detecting fraudulent transactions
## Overview
This project is a machine learning model built to detect fraudulent transactions.  
It analyzes transaction data and predicts whether a transaction is **fraudulent or legitimate** using supervised learning techniques.

## Objective
To build a system that can accurately identify fraud patterns in financial transaction data and improve detection efficiency.

## Features
- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Feature engineering  
- Model training and evaluation  
- Fraud prediction using machine learning  

## Tech Stack
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Matplotlib / Seaborn  
- Jupyter Notebook  

## Model Used
A classification model was trained on transaction data to detect fraudulent activity.  


## Project Workflow
1. Load dataset  
2. Clean and preprocess data  
3. Perform exploratory data analysis  
4. Train machine learning model  
5. Evaluate model performance  
6. Make predictions  

## How to Run

1. Clone the repository
2. Open the notebook in Jupyter or Google Colab
3. Install required libraries using:
   pip install -r requirements.txt
4. Run all cells
